# CircleHome

## Description
CircleHome is a mobile app that brings household task management and family caregiving into one shared space. Users can create and assign chores, appointments, medications, and errands for everyone in their household, with tasks automatically distributed based on each member's availability and capacity that week. Instead of juggling a chore app, a group chat, and a notes app simultaneously, CircleHome keeps everything in one feed.

The app is built around two core use cases that are almost always happening at the same time. The first is everyday household coordination, where tasks rotate fairly among members and a shared team health score shows how balanced the load is without anyone having to point fingers. The second is family caregiving, where members can log quick notes after visiting a loved one covering mood, appetite, and symptoms. These notes build into a running health log that gets summarized weekly and can be exported as a PDF to share with a doctor.

To prevent care tasks from slipping through, CircleHome includes an escalation system that notifies a backup person if a task goes unclaimed, and can optionally reach out to the care recipient directly. This project focuses on building a full-stack mobile application with an emphasis on clean UI, real-time updates, and genuinely useful automation. The goal is to reduce the invisible coordination load that falls on families managing a home and a loved one at the same time.
